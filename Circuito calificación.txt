Circuito de calificación:

El usuario califica una publicación o comentario. El usuario solamente
puede calificar como positivo o negativo.

Cuando el usuario califica una publicación o comentario, se crea una
calificación asociada a este. Esta calificación a su vez posee un 
puntaje. Este puntaje se calcula automáticamente, es parte de la lógica
del método crear de la clase puntaje.

El puntaje se calcula así:
	Si el promedioCalificaciones del usuario está entre 0 y 1 entonces puntaje =1
 	Si el promedioCalificaciones del usuario es entre 2 entonces puntaje =2
	Si el promedioCalificaciones del usuario es entre 3 entonces puntaje =3
	Si el promedioCalificaciones del usuario es entre 4 entonces puntaje =4
	Si el promedioCalificaciones del usuario es mayor 5 entonces puntaje =5

Como dije antes, el usuario puede calificar positivo o negativo.
Ahí es donde entra en juego el signo del puntaje.
